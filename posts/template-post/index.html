Exemplo de texto com marcações

O pacote dados (Quiroga et al. 2022) disponibiliza a base de dados pinguins, uma versão traduzida para português do pacote palmerpenguins (Horst, Hill, e Gorman 2020). A tradução dessa base foi feita por Jean Prado, que faz parte da co-organização da R-Ladies São Paulo!

Além destes pacotes, nesse arquivo foram usados o knitr (Xie 2022) e tidyverse (Wickham 2022).

Carregando pacotes

library(tidyverse)
library(dados)

Exemplo de código em linha

A base pinguins apresenta 344 pinguins (pois cada linha representa um pinguim). As colunas presentes na base são: especie, ilha, comprimento_bico, profundidade_bico, comprimento_nadadeira, massa_corporal, sexo e ano.

Exemplo de gráfico

pinguins |> 
  ggplot() +
  aes(x = massa_corporal, y = comprimento_nadadeira, color = especie) +
  geom_point() +
  theme_light() + 
  scale_color_viridis_d()

Exemplo de tabela

pinguins |> 
  count(especie, ilha, name = "quantidade") |> 
  knitr::kable()
